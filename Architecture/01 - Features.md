# Circlo Features

## Core Features

### 1. Listings Marketplace

Students can create and browse listings for things they want to sell, offer, or share.

Listing categories:
- Items for sale
- Services
- Tickets
- Quick gigs
- Rentals

Examples:
- Selling a used bike
- Selling textbooks
- Offering tutoring
- Offering moving help
- Renting out a calculator, camera, or equipment
- Selling event tickets
- Posting a quick paid task

---

### 2. Service Offering

Students can offer useful services to other UIC students.

Examples:
- Tutoring
- Homework help
- Resume review
- Moving help
- Photography
- Haircut / beauty service
- Tech help
- Renting personal items or equipment

---

### 3. Community Feed

A social feed where students can post quick updates, needs, questions, or offers.

Post types:
- Looking for
- Offering
- Question
- General post

Examples:
- “Looking for someone selling a CS textbook”
- “Offering tutoring for CS 251”
- “Does anyone know a good place to fix a bike?”
- “Selling one ticket for this weekend”

---

### 4. Swipe Discovery

A Tinder-style discovery flow for marketplace listings.

User actions:
- Swipe right / like
- Swipe left / skip
- Save interest in a listing
- Use interactions later for recommendations

Purpose:
- Make browsing faster
- Help students discover items/services quickly
- Store preferences for future ranking

---

### 5. Real-Time Chat

Students can message each other directly after finding a listing, service, or post.

Chat features:
- Create conversations
- Send messages in real time
- Store message history
- View all conversations
- Continue conversations later

Use cases:
- Ask if an item is still available
- Negotiate price
- Coordinate meetup time
- Discuss service details

---

## MVP Features

For the first version, focus on:

1. UIC student authentication
2. Create and browse listings
3. Search and filter listings
4. Upload listing images
5. Real-time chat
6. Basic community feed
7. Like / skip interactions

---

## Future Features

Possible future improvements:
